Cheatsheets Collected from My Reports

ffuf syntax collected from my WebGoat and web-fuzzing reports:

./ffuf -w <path-to-wordlist> (-X <HTTP METHOD> -d <method-data> -H <method-headers> -x <http(s)://proxy-url:port>)

Wfuzz, Hydra, hashcat and john syntax from my password cracking report:

wfuzz -w <path-to-wordlist> (-d <POST DATA>) <url>

 

hydra -l <single username> / -L <username list> -p <single password> / -P <password list> <host> -s <port> <http-form-type>
"<endpoint>:<form>(eg. username=^USER^&password=^PASS^:<error-message> -V (verbose mode)

 

./hashcat64.exe –force -a <type-of-attack> -m <hashtype> -o <outputfile> <hashfile> <wordlist> (--self-test-disable)

 

(sudo) john <hashfile> --wordlist=<FILE>

Cheatsheets Collected from Coursemates Reports

Python pty shell from Niko Heiskanen:

(After getting an initial shell):

python(3) -c 'import pty; pty.spawn("/bin/bash")'

CTRL + Z

ssty raw -echo

fg

Enter

(back in shell)

export TERM=<terminal-type>

Cracking ssh with medusa from Caius Juvonen:

medusa -h <host-ip> -u <username> -P <path-to-wordlist> -M <module, for example ssh> -O <outputfile> -t <attempts-per-second>

Scanning for WordPress vulnerabilities from Arttu Talvio:

First scan for possible WordPress endpoints:

dirb <host>

Then:

wpscan -v (verbose) –-url <host-address/wordpress-endpoint>
